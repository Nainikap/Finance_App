# 💜 Finance Tracker

A high-fidelity, **Glassmorphic** personal finance management system. This application uses a deep-space purple aesthetic, leveraging translucent layers and background blurs to create a premium, modern user experience.

---

## 🚀 Features

### 📊 Financial Dashboard
* **Digital Wallet & Cards:** Sleek, glass-styled overview of your Master Card and Wallet balances.
* **Data Visualization:** Integrated Line and Donut charts for real-time tracking of savings vs. expenses.
* **Transaction Feed:** A quick-glance sidebar for recent receipts and payables.

### ➕ Expense Management
* **Dynamic Form:** Log expenses with names, amounts, dates, and "Paid/Unpaid" status.
* **File Support:** Ability to upload and track digital invoices.
* **Live History:** Categorized view of "New" vs. "Past" expenses with hover-effect rows.

### ⚙️ Account Settings
* **User Profile:** Glowing avatar generated from user initials.
* **Information Control:** Manage general info, currency preferences, and notification toggles.
* **Security:** Clean, responsive layout for account maintenance and logout functionality.

---

## 🎨 Design System

### The Aesthetic: Glassmorphism
The project uses a custom-built CSS framework based on:
* **Background:** `linear-gradient(135deg, #0f0a19 0%, #210821 100%)`
* **Surface:** `rgba(255, 255, 255, 0.05)` (Low opacity white)
* **Blur:** `backdrop-filter: blur(15px)`
* **Accents:** `#cbb2e5` (Soft Lavender)
